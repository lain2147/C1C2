# 📸 BOM表拍照建议

为了获得最佳的OCR识别效果，请按照以下建议拍摄BOM表：

## ✅ 正确的拍照方式

### 1. **位置和角度**
- 📐 **垂直拍摄**：相机镜头垂直对准表格，避免倾斜
- 🎯 **居中对齐**：将整个表格放在画面中央
- 📏 **保持距离**：确保整个表格都在画面内，但文字要足够大

### 2. **光线要求**
- 💡 **均匀照明**：使用均匀的白光
- 🚫 **避免阴影**：不要让手机或身体挡住光源
- ✨ **避免反光**：调整角度避免纸张反光
- 🌞 **自然光最佳**：在自然光下拍摄效果最好

### 3. **对焦和清晰度**
- 🔍 **对焦准确**：点击屏幕对焦到文字区域
- 📱 **保持稳定**：拍摄时手机要稳定，避免抖动
- 📏 **适当距离**：太近会模糊，太远文字太小

### 4. **背景和对比度**
- ⬛ **深色背景**：在深色桌面上拍摄白色纸张
- 📄 **平整摆放**：确保纸张平整，没有褶皱
- 🎨 **高对比度**：黑色文字 + 白色纸张 = 最佳效果

## ❌ 常见错误

### 错误示例1：角度倾斜 ❌
```
结果：文字变形，识别率低
```

### 错误示例2：光线不足 ❌
```
结果：文字模糊，识别困难
```

### 错误示例3：反光严重 ❌
```
结果：部分文字看不清
```

## 📱 分段拍摄技巧

如果表格很大，建议**分段拍摄**：

### 步骤1：规划区域
```
把表格分成2-3个区域：
┌─────────┐
│  区域1   │ ← 上半部分
├─────────┤
│  区域2   │ ← 中间部分
├─────────┤
│  区域3   │ ← 下半部分
└─────────┘
```

### 步骤2：逐个拍摄
- 每个区域单独拍一张清晰的照片
- 确保区域之间有少许重叠（防止遗漏）

### 步骤3：批量上传
- 在工具中一次上传所有照片
- 系统会自动合并识别结果

## 🎯 检查清单

拍照前请确认：
- [ ] 相机垂直对准表格
- [ ] 光线充足且均匀
- [ ] 没有阴影遮挡
- [ ] 对焦清晰
- [ ] 文字大小适中（不要太小）
- [ ] 整个区域都在画面内

## 💡 专业建议

### 使用扫描仪（最佳）
如果可能，使用扫描仪扫描BOM表：
- ✅ 清晰度最高
- ✅ 没有角度问题
- ✅ 光线均匀
- ✅ 识别率接近100%

### 使用文档扫描APP
推荐使用：
- Microsoft Office Lens
- Adobe Scan
- CamScanner

这些APP会自动：
- 矫正角度
- 增强对比度
- 去除阴影
- 裁剪边缘
